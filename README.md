# Pebble

![Go](https://img.shields.io/badge/Go-00ADD8?logo=go&logoColor=white) ![Svelte](https://img.shields.io/badge/Svelte-FF3E00?logo=svelte&logoColor=white) ![Cloudflare Tunnel](https://img.shields.io/badge/Cloudflare%20Tunnel-F38020?logo=cloudflare&logoColor=white)

Pebble is a modern, lightweight homelab Caddyfile manager with a Go backend and a Svelte UI for a clean, fast in-browser experience.
It keeps Caddy configuration editing and file browsing simple and responsive without the overhead of a full IDE.
The container includes a Cloudflare Tunnel so it can be safely exposed beyond your local network with minimal setup.

Highlights: Caddyfile manager, Go + Svelte, Cloudflare Tunnel, homelab-focused.
